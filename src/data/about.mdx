# Where did this come from?

It all started with an episode of the [APIs You Won't Hate Podcast](https://share.transistor.fm/s/789a4970), where [Matt Trask](https://twitter.com/matthewtrask) and [Mike Bifulco](https://twitter.com/irreverentmike) discussed how challenging it's been to get folks to adopt the name `Open API` instead of `Swagger`.
